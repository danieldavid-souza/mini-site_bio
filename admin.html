Painel Administrativo

Gerencie os produtos do seu mini site. Após editar, exporte o arquivo data.js e substitua o antigo no seu projeto.

Instruções:

  1. Use os campos para editar os produtos. As alterações são salvas localmente no navegador.
  2. Clique em "Adicionar Produto" para criar um novo item.
  3. Clique em "Salvar Alterações" para confirmar as mudanças na tabela.
  4. Quando terminar, clique em "Exportar para data.js".
  5. Salve o arquivo baixado com o nome data.js dentro da pasta do seu projeto, substituindo o antigo.
  6. Para carregar dados de outro catálogo, use o botão "Importar JSON".
